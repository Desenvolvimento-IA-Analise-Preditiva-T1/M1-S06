# Material das auldas da semana 06

Este repositório contém materiais e exercícios guiados utilizados em aula. O objetivo é auxiliar no aprendizado prático utilizando notebooks Jupyter.  

**Observação**: Existem muitos códigos de rascunho que, possivelmente, farão sentido com o vídeo da aula.

## 📂 Estrutura do Projeto

```bash
.
├── aulas/
│   └── aula01.ipynb
│   └── aula02.ipynb
├── .gitignore
├── README.md
└── requirements.txt

```

## 🚀 Como utilizar

### 1. Clone o repositório

```bash
git clone <URL_DO_REPOSITORIO>
cd <NOME_DO_REPOSITORIO>
```

### 2. Crie um ambiente virtual (opcional, mas recomendado)

**Podem fazer essa etapa diretamente pelo **VS Code**, como vimos em aula, ou seguir desta forma:**

#### Linux / macOS

```bash
python -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Instale as dependências

```bash
pip install -r requirements.txt
```

### 4. Execute o VS CS Code

Depois disso, abra o arquivo:

```bash
exercicios/exercicios_guiados.ipynb
```

## 📘 Conteúdo

Os notebooks contêm o conteúdo das aulas, além de exercícios guiados (resolvidos em aula) para fixação dos conceitos apresentados.
